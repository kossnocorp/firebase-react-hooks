## Installation

The library is available as an npm package. To install it run:

```sh
# React:
npm install firebase-react-hooks --save
# Or using Yarn:
yarn add firebase-react-hooks

# Preact:
npm install firebase-preact-hooks --save
# Or using Yarn:
yarn add firebase-preact-hooks
```
